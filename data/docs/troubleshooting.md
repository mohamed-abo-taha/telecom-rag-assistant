# Troubleshooting

## No mobile data
1. Check that mobile data is enabled and the device is not in Airplane mode.
2. Confirm the plan has remaining data in the app under Usage.
3. Toggle Airplane mode on and off, then restart the device.
4. If it still fails, reset network settings (this clears saved Wi-Fi passwords).

## Data not working abroad (roaming)
- Roaming is off by default. Enable it in the app under Settings > Roaming.
- Only the Unlimited Pro plan includes roaming data; other plans need a roaming pass at 5 USD/day.
- If roaming is on and data still fails, manually select the partner network in device settings.

## Slow data speed
- Speeds drop to 5 Mbps on Unlimited Basic after 100 GB in a cycle; this resets the next cycle.
- Congestion in busy areas can reduce speed temporarily.
- Run a speed test from a different location to compare.
